# Projektantrag <br> "Create your sounds"

## USP
Auf dieser Website soll es möglich sein, Sounds abzuspielen, eigene Sounds hochzuladen, zu erstellen und evtl. aufzunehmen

## UI & UX
Eine Seite ist mit verschiedenen Buttons, die Sounds abspielen, gefüllt.<br>
Auf einer weiteren Seite soll es möglich sein, Sounds aufzunehmen/hochzuladen, das aber nur mit einem erstellten Account

## Coder Plan
**Technologien:**
- HTML
- CSS
- JS
- PHP
- SQL
- Node.js

**Datenbank:**
- Tabelle mit Usern
- Tabelle mit Sounds
- Tabelle mit gespeicherten Aufnahmen (Link zur Festplatte)
- Tabelle mit verschiedenen Soundboards/Geräten
